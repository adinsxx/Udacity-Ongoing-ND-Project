Lesson 1: Basics of the Web

The Web

The world wide web is comprised of a collection of HTML (HyperText Markup Language). Invented in the early 1990s, it now hosts close to 30 million HTML documents and is comprised of mainly HTML documents.

How It Works

The web is comprised of many computers called 'servers'. These computers only serve the function of storing files; unlike a PC, which is primarily used to read the files off these servers. For more information on how computers get that information, check out this link

What Is HTML?

HTML (or HyperText Markup Language) form the majority of content on the web, as stated earlier. HTML is text content, which makes up pretty much all text you see on a webpage.

Tags and Elements; Inline vs. Block

Tags in HTML define the text placed in them. Tags can bold and italicize text, and even be used to create links out of certain pieces of text. Lesson 2: Structured Documents in HTML

Utilizing Browser Tools

As seen in the previous lesson, HTML elements are either inline or block. Block elements form a 'box' around the content inside of them. The box itself is invisible, but that whatever is inside the box can be manipulated.

Tree Struture and HTML

Similar to a family tree, the 'tree structure' of HTML comes from the fact that HTML elements can have other HTML elements nested inside of them. In a browser, this can show up as a series of nested boxes (yep, boxes on boxes) and the boxes will have boxes inside boxes inside of boxes. Indentation is also a good way to note the nesting of HTML elements. It can be defined by the 'wave' on the left side of the document.
